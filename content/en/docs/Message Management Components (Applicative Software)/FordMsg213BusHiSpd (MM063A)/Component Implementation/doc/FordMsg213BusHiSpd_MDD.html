FordMsg213BusHiSpd & High-Level Description

The purpose of the Ford Message 213 Bus High Speed function is to provide the Electric Power Steering system with signal values for the Brake Oscillation Reduction, Lane Detection Warning, LKA, Traffic Jam Assist and Lane Center Assist functions from CAN. The Ford Message 213 function will perform the missing message and signal invalid diagnostics as well as provide a validity flag for the signal values and received message.
