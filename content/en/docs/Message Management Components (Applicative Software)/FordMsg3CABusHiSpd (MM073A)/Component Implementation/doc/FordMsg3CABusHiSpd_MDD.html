FordMsg3CABusHiSpd & High-Level Description

The purpose of the Ford Message 3CA Bus High Speed function is to provide the Electric Power Steering system with signal values for Lane Detection Warning and Lane Keep Assist functions form CAN. The Ford Message 3CA function will perform the missing message and signal invalid diagnostics as well as provide a validity signal for the signal values and received message.
