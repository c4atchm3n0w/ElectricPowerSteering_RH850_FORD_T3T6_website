FordMsg216BusHiSpd & High-Level Description

The purpose of the Ford Message 216 Bus High Speed function is to provide the Electric Power Steering system with signal values for the Trailer Back Up Assist, Torque Steer Compensation, and Brake Oscillation Reduction functions from CAN. The Ford Message216 function will perform the missing message diagnostic as well as provide a validity signal for the received message.
