FordMsg083BusHiSpd& High-Level Description

The purpose of the Ford Message 083 Bus High Speed function is to provide the Electric Power Steering system with signal values for the Lane Detection Warning function from CAN. The Ford Message 083 function will perform the missing message diagnostics as well as provide a validity flag for the signal values and received message. Detailed requirements have implemented the Ford DBC file for guidance.
